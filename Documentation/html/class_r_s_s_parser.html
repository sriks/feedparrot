Detailed Description

RSSParser is a special xml parser for RSS feeds. It is based on XQuery as backend. It is very light weight as it does not own the source data. Using RSSParser requires some basic knowledge of RSS elements. It suports RSS Specification http://www.rssboard.org/rss-specification.

Setting source

RSS Source can be set using setSource.

    QRSSParser parser;
    QFile sourceFile("rss.xml"); // file in which rss feed is collected/stored.
    if(source.open(QIODevice::ReadOnly))
    {
       // set source
       parser.setSource(sourceFile);

       // use parser
    }
    sourceFile.close();


or

QBuffer can also be used in setSource. 

    QString sourceXml;
    QBuffer buffer(sourceXml.toUtf8());
    if(buffer.open(QIODevice::ReadOnly)) // ensure buffer is opened in read mode
    {
        parser.setSource(&buffer);
        if(parser.isValid())
        {
        // use parser
        }
    }
    buffer.close();

Usage

Usage of RSSParser is designed around RSS Specification http://www.rssboard.org/rss-specification

  • channelElement is used to query channel elements
  • itemElement is used to query item elements
  • some special methods like imageUrl, itemCount, category, categories can be used to query other elements.
    RSS elements can be specified using RSSParser::RSSElement 
        QRSSParser parser;
    QFile sourceFile("rss.xml"); // file in which rss feed is collected/stored.
    if(source.open(QIODevice::ReadOnly))
    {
       // set source
       parser.setSource(sourceFile);

       if(parser.isValid())
       {
       // get name of channel
       QString channelname = parser.channelElement(RSSParser::title);

       // get channel image
       QUrl imageUrl = parser.imageUrl();

       // get count of items
       int count = parser.itemCount();

       // get title of first item
       QString title = parser.itemElement(0,RSSParser::title);
       }
    }
    sourceFile.close();

Error Handling

Parsing error in the backend (XQuery) can be found using isError.

    if(parser.isValid())
    {
        // querying out of bounds item
        QString title = parser.itemElement(35,RSSParser::title);

        if(parser.isError())
        {
            // some error in request.
        }
    }

isError is reflects error state of the latest request only.
